# dcac-temporal-drift-embed

One-container Labtainer lab for video steganography.

## Student workflow

1. Run `ffmpeg -hide_banner -i input.mp4`.
2. Run `python3 tao_tin.py`.
3. Confirm `output.avi`.
4. Run `python3 tach_tin.py`.

## Grading

The grader uses four lightweight transcript checks and does not rerun video processing.
